# Fabs Kubernetes Installer - Ansible based
--------------------

AIO Kubernetes installer (say no to minikube) which can be deployed on bare metal or VMs.

## Supported platforms:

* Debian based
* RPM based (Fedora, CentOS, etc)

## Tested platforms:

* Fedora 32
* Debian
